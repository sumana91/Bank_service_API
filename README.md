## Bank Service APIs 
Application to do CRUD operations on a bank system using Nodejs and Postgresql. 

### Requirements
- Nodejs
- Express js
- Postgresql 
- Postman
- Knex 
- Bookshelf

### Install all the dependencies 
- `npm install --save`

### Run the migration script
- `knex migration:latest` from the migration folder. 
- Setup the database
